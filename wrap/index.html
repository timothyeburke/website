Hi there!

 

The page content goes here. This page content will be pushed down as the header row gets taller when the text wraps. This is accomplished without any JavaScript height measurement. If you adjust the width of your browser window, the text will wrap in and the heaer row will get taller or shorter. Using a numberic margin or position for the page content can result in the content being covered by the fixed header bar.

 

To achieve a HTML/CSS-only workaround, place an exact copy of the header bar underneath of it, but not fixed. That includes the same height, padding, margin, etc., and most importantly the same text. This way, as the width of the page varies, resulting in different heights for the fixed header bar, the hidden div behind it will adjust to the same height, pushing the content down. When the page is narrow enough to scroll, the content will scroll under the header bar, and will always be adjusted down far enough when scrolled to the top.
