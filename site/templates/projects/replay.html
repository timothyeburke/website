Replay Web Prototype

Technologies used: JavaScript, Cesium, Sass

This rapid prototype was a proof of concept effort for building a time-variable, geospatial data playback tool in a web browser. The basic featureset needed to mimic an existing desktop application used for playing back similar data, also produced by Ringtail Design. The globe and map tiles are rendered using Cesium, an open-source WebGL globe and map tile rendering engine.
