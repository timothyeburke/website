Intelligence Needs Kiosk

Technologies used: JavaScript, Knockout (1st rev.), AngularJS (2nd rev.), Sass, node.js, Express, MongoDB

Design and implement an initial prototype of a touch-screen kiosk view of intelligence needs for Ringtail Design, allowing users to walk up, select different questions, and see artifacts related to them. Initial prototype was used to conduct user sessions, leading to a revised design and implementation. First revision was built using Knockout and second version was built using AngularJS. The data service is node.js, Express, and MongoDB.
